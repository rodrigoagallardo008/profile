# Hi, I'm Rodrigo!
I'm a computational designer and researcher at MIT working at the intersection of HCI, AR/VR, and design. I build interactive systems and tools focused on spatial computing and human performance.

**A few things I've been making**
* VR learning environments for queueing theory — MIT AeroAstro
* AR tools for computational fabrication tasks
* Haptic XR user studies on spatial perception — Rice MAHI Lab
* Parametric skate feature design based on motion and rhythm
